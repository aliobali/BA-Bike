KEY FILES TO KNOW
─────────────────────────────────────────────────────────────────────
BikeUdpReceiver.cs
  ├─ Listens on ports 5005 (steering) and 5006 (speed)
  ├─ Parses labeled and unlabeled UDP messages
  └─ Properties: Speed, SteeringAngleDegrees

RayzNGamesInputAdapter.cs
  ├─ Converts speed (m/s) to vertical input (0-1)
  ├─ Converts angle (°) to horizontal input (-1 to 1)
  ├─ Applies smoothing for natural feel
  └─ Implements auto-braking

HUDManager.cs
  ├─ Displays sensor input values
  ├─ Shows virtual bike response
  └─ Optional debug information

ObjectFollower.cs
  ├─ Third-person camera following
  └─ VR-compatible camera rig support
